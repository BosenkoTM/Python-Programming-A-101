Для работы со всеми интерфейсами (на базе PySide6 и Tkinter) вам понадобятся следующие библиотеки:

### PySide6 (Qt6)
Эта библиотека устанавливается через менеджер пакетов `pip`. Выполните команду в терминале PyCharm (вкладка **Terminal** внизу экрана):

```bash
pip install PySide6
```
*(Для macOS или Linux может потребоваться команда `pip3 install PySide6`)*

### Tkinter
Библиотека **Tkinter** является стандартным встроенным инструментом Python, поэтому её **не нужно** устанавливать через `pip`:

* **Windows и macOS:** Она уже предустановлена вместе с интерпретатором Python. Никаких дополнительных действий выполнять не требуется.
* **Linux (Ubuntu/Debian):** В некоторых дистрибутивах Linux она может отсутствовать по умолчанию. Если при попытке импорта возникает ошибка, установите её через системный терминал (не в PyCharm):
  ```bash
  sudo apt-get install python3-tk
  ```
